# Farkle Scorer Pro
A simple score keeping app for Farkle. Download here: [https://farkle-scorer.web.app](https://farkle-scorer.web.app)

## Features
 🎲 Unlimited Players  
 🎲 Custom player names  
 🎲 Rolling 3 one's worth 1000 points (optional)  
 🎲 Scoreboard
